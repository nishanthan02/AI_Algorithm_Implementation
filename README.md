# 🎮 Chaos Tic Tac Toe

A modern Tic Tac Toe game built with **Python** and **Tkinter**, featuring a cyberpunk-inspired neon UI, smooth animations, and an unbeatable AI powered by the **Minimax Algorithm**.

---

## ✨ Features

- 🤖 Single Player mode against an AI
- 👥 Two Player mode
- 🧠 Unbeatable AI using the Minimax algorithm
- 🎨 Neon Cyberpunk UI
- ✨ Animated glowing effects and title flicker
- 🏆 Live Scoreboard
- 🎉 Funny victory messages
- 🔄 One-click game restart
- 💡 Interactive hover effects
- ⚡ Winning line animation

## 🚀 Installation

Clone the repository

```bash

```

Go to the project directory

```bash
cd AI_Algorithm_Implementation
```

Run the application

```bash
python app.py
```

---

## 🎮 How to Play

### Single Player
- Select **AI Mode**
- Choose your symbol (X or O)
- Play against the computer.

### Two Player
- Select **2 Player Mode**
- Players alternate turns until someone wins or the game ends in a draw.

---

## 🧠 AI Strategy

The AI uses the **Minimax Algorithm**, which evaluates every possible move and always selects the optimal one. This makes the AI impossible to defeat if it plays perfectly.
